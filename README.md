# Analise-de-dados-Residencia
Scripts e documentos relevantes relacionados à residência em análise de dados 2025 - Ecoa-PUC Rio, sob mentoria de Mateus Levi.

## Projeto sobre Desigualdade Social:

Foram analisadas diversas fontes de dados, tranformadas em gráficos para melhor visualização e complementadas com textos e imagens sobre o tema, seguindo padroes de design e storytelling.

## Competencias adquiridas na Residência:

Google Looker Studio;
Visualização de dados: utilização de diferentes tipos de gráficos;
SQL (consultas, joins);
Python (pandas para tratamento de dados);
Big Query (ETL de dados através de consulta SQL ou script Python no Colab).
Design (Style Guide e Wireframe);
Storytelling (contexto, processo, descobertas e conclusões).
